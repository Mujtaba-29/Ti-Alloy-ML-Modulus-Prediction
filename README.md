# Ti-Alloy-ML-Modulus-Prediction
ML-driven framework for predicting the Young’s modulus of multi-component Ti-alloys. By integrating thermodynamic descriptors (entropy, Mo-eq) with an optimized XGBoost model, this project provides a computational tool to accelerate the discovery of low-modulus biomedical materials and prevent stress shielding in implants. Includes code &amp; dataset.
